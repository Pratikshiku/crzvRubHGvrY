### 三、功能描述
系统分为（身份）：教师、学生家长、管理员。

**教师功能：**
1. 登录
2. 修改密码
3. 个人信息
4. 公告管理
5. 班主任通知管理
6. 缴费管理
7. 家长学生管理、考试管理、成绩管理、意见建议管理
8. 课表管理
9. 作业管理、回答管理


**学生家长功能：**
1. 登录
2. 修改密码
3. 个人信息
4. 公告管理
5. 班主任通知管理
6. 缴费管理
7. 家长学生管理、考试管理、成绩管理、意见建议管理
8. 课表管理
9. 作业管理、回答管理




**管理员功能：**
1. 登录
2. 个人中心(修改密码、个人信息）
3. 基础数据管理（班级管理、班主任通知类型管理、公告类型管理、管理员通知类型管理、缴费类型管理、科目管理、意见建议类型管理、作业类型管理）
4. 公告管理
5. 通知管理（管理员通知管理、班主任通知管理）
6. 缴费管理
7. 教师管理
8. 家长学生管理、成绩管理、考试管理、意见建议管理
9. 课表管理
10. 作业管理、回答管理
